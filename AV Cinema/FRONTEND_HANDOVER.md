# AV Cinema — 前端功能清单

## 1. 影片列表页（/category/:category）

**路径**：`/category/all`、`/category/AV`、`/category/动漫`、`/category/欧美`、`/category/国产`

**功能**：

| 功能 | 说明 |
|---|---|
| 分类浏览 | 侧栏点击分类，展示该分类下全部影片（去重后） |
| 统计卡片 | 页面顶部显示该分类影片数、总女优数、总收藏数，居中排列 |
| 排序 | 支持按加入日期、发行日期、女优名、播放次数排序，可切换正序/倒序/随机 |
| 标记筛选 | AV 分类专属：勾选"中字/破解/无码"筛选项 |
| 影片卡片 | 每个卡片显示封面图、分类标签、标记（中字/破解/无码）、片名、番号、女优名 |
| 番号后缀剥离 | 卡片上番号不显示 `-C/-U/-UC` 后缀 |
| 右键菜单 | 右键卡片弹出"删除影片"和"删除影片（含文件）" |
| 无限滚动 | 每次加载 24 部，滚动到底部自动加载更多 |
| 缩略图缓存 | 图片使用 LRU 缓存（上限 500 张），首次加载后秒开 |
| 收藏高亮 | 已收藏的影片番号变红，已收藏的女优名字变红 |

---

## 2. 影片详情页（/movie/:id）

**功能**：

| 功能 | 说明 |
|---|---|
| 番号+片名 | 页面最上方显示分类标签、收藏按钮、片名、番号 |
| 番号复制 | 点击番号文字自动复制到剪贴板（展示时已剥离 `-C/-U/-UC` 后缀） |
| 图片轮播 | 显示截图列表，点击切换大图，编辑模式下可拖拽排序、删除、新增截图 |
| 基本信息 | 番号、女优、发行日期、时长、系列、观看次数 |
| 女优点击跳转 | 点击女优名跳转到女优详情页 |
| 时长格式化 | 秒自动转为 `Xh Ym` 或 `X 分钟` |
| 标签 | 按女优标签（红色）和影片标签（蓝色）分开显示，按分类顺序+位置排序 |
| 播放按钮 | 有视频文件时显示"播放影片"按钮，调用系统播放器 |
| 编辑模式 | 点击"编辑"进入，可改片名/番号/发行日期/女优/时长（分钟数）/系列/标记 |
| 标记开关 | 中字/破解/无码 三个开关 |
| 标签编辑 | 编辑模式下可添加/删除标签，支持预设标签快捷添加 |
| 翻译片名 | 调用 DeepSeek API 翻译片名，有撤回功能 |
| 收藏 | 点击星标切换收藏状态 |

## 3. 女优列表页（/actresses）

**功能**：

| 功能 | 说明 |
|---|---|
| 搜索 | 按名字、中文名、别名搜索 |
| 分类筛选 | AV / 欧美 / 国产 |
| 作品状态筛选 | 有作品 / 无作品 |
| 排序 | 作品数 / 年龄 / 罩杯，可切换正序/倒序 |

| 无限滚动 | 每次加载 24 人，滚动到底自动加载 |
| 右键删除 | 右键女优卡片弹出删除选项 |
| 批量翻译 | 弹窗显示进度条，调用 DeepSeek API 翻译全部女优名 |

## 4. 女优详情页（/actress/:id）

**功能**：

| 功能 | 说明 |
|---|---|
| 头像 | 圆形大图，点击可更换 |
| 信息显示 | ID、名字、别名、分类标签、出生日期、身高、三围、罩杯 |
| 标签 | 显示女优标签（来自 `bz` 字段），可编辑 |

| 收藏 | 星标切换收藏状态 |
| 编辑模式 | 可编辑名字/别名/出生日期/身高/罩杯/三围/标签，保存后立即刷新 |
| 统计 | 作品数量和评分 |

## 5. 添加影片页（/add）

**功能**：

| 功能 | 说明 |
|---|---|
| 批量刮削 | 拖拽或点击选择多个视频文件 -> 自动提取番号 -> 刮削 JavDB -> 移动文件 -> 入库 |
| 刮削进度 | 显示 "刮削中... X/N" 计数器，完成显示结果（✓/✗） |
| 手动添加 | 填写番号/片名/女优/分类/发行日期/系列等信息后保存 |
| 手动选择文件 | 独立选择封面图、预览图（多选）、视频文件 |
| 番号提取 | 自动从文件名提取番号，支持 FC2、欧美日期、无码数字、标准 JAV 多种格式 |
| 添加女优 | 切换到"添加女优"标签，填写名字/别名/三围/罩杯/身高/生日/出道日等信息 |

---

## 6. 搜索（顶部标题栏 + /search 页面）

| 功能 | 说明 |
|---|---|

## 7. 标签浏览页（/tags）

**功能**：

| 功能 | 说明 |
|---|---|
| 分类展示 | 按分类（主题/角色/服装/体型/行为/玩法/其他/口碑 + 自定义）展示标签 |
| 点击筛选 | 点击标签筛选包含该标签的影片 |

| 删除分类 | 编辑模式下可删除分类（标签数据不丢失） |

## 12. 全局规则

| 规则 | 说明 |
|---|---|
| 番号后缀 | 数据库保留 `-C/-U/-UC`，全软件展示时剥离 |
| 女优显示名 | 任何地方显示的是 `femas.mz`（显示名），不是 `bm`（别名） |
| 图片加载 | 全部走 `local://` 自定义协议，不再使用 base64 |
| 分类颜色 | AV=红、动漫=紫、欧美=蓝、国产=金 |
| 侧栏计数 | 按 `py`（视频路径）去重统计 |
| 标签格式 | 数据库存储带中文逗号包围，前端显示去掉逗号 |
